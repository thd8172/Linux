ls 디렉토리 목록 변경
ls -a 모든파일 열기 숨겨진파일도 열기
ls -최근파일부터 출력

cp 파일의 내용을 다른파일에 복사
cp -a 원본파일을 유지하면서 복사
cp -b복사할 대상을 덮어쓰거지 지울떄 백업하는 파일 만듬
cp -f 복사할 파일이 존재할떄 삭제하고 복사
cp-r 디렉토리내에 있는 모든 파일을 통째로 복사

mv 파일이름을 변경
mv -b 파일이 지워지기 전에 백업파일 만듬
     -f 이동할 디렉토리에 이미 파일이 존재하는 경우 기존 파일은 없어지고 강제로 새로운 파일이 이동
    -i이미 파일이 존재하는 경우 이동여부를 확인
  -u 대상 파일보다 원본 파일이 최신것일 때 업그레이드
 -v 파일 옮기기 전의 과정을 보여줌

rm 지정한 파일 또는 디렉토리를 삭제한다
  rm -d 디렉토리 삭제
       -f 삭제할 것인가 메세지 없이 강제 삭제
      -r 디렉토리 자신을 포함 모든 파일과 하위 디렉토리를 삭제
mkdir 디렉토리 파일을 생성 해주는 명령어
rkdir   빈디렉토리 삭제 
rkdir -p 지정한 하위 티렉토리까지 한번에 삭제

pwd 작업 디렉토리 출력
