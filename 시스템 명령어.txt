시스템 명령어

who의 옵션
 -i     rdle time과 함께 사용자 출력
-m   who명령을 실행한 사용자 표시
-q  사용자 이름과 사용자 수 출력
-w -t 각 사용자 메세지 설정 상태 출력

cat의 옵션
e  -vE 옵션과 같음
b 공백라인을 제외한 모든 라인에 행 번호를 붙임
n 출력되는 파일 내용들에 대해서 번호를 붙임
t -vt 옵션과 같음

입출력 반향 전환
입출력 redirection 기호

 <  (입력 리디렉션) 키보드가 아닌 다른 파일이나 장치로부터 입력을 받을수 있음
 > (출력 리디렉션) 출력결과를 화면이 아닌 다른 파일이나 프린터, 통신 포트와 같은 다른장치로 출력
>> (추가 리디렉션) 기존의 파일이 존재하면 기존 내용 뒤에 새로운 결과 출력을 추가한다
|  (파이프) 한 프로그램의 표준 출력을 다른 프로그램의 표준 입력으로 직접 연결


clear 화면을 깨끗이 지워주는 명령어

df 하드디스크의 용량이 얼마나 남아있는지 확인하는 명령어

du 사용자 디스크 용량에 대한 정보 제공

find 원하는 파일 위치 찾기

more 화면 단위로 파일 내용 출력

head 파일의 앞부분 몇 라인 출력

tail 파일의 마지막 몇 라인 출력

grep 지정된 파일에 대해서 일치하는 라인 출력
